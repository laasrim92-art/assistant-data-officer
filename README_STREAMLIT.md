# Assistant Data Officer

Placez les fichiers dans le même dossier, puis exécutez :

```bash
pip install -r requirements_streamlit.txt
streamlit run app_streamlit.py
```

L'application prédit le statut du dossier, affiche un niveau de confiance, explique les points à contrôler et génère un brouillon d'email lorsque des informations bloquantes sont absentes.
